Manual de Usuario de ComerciPlus

Este manual de usuario ha sido diseñado para proporcionar una guía completa y detallada sobre cómo utilizar todas las funcionalidades del software ComerciPlus. Su objetivo es ayudarle a entender y aprovechar al máximo las capacidades del sistema, facilitando así la gestión de su negocio de manera eficiente y efectiva.

Módulo de Categoría de Productos

El módulo de categoría de productos sirve para darle un orden a los productos y así mantenerlos de manera ordenadas

90. Seleccionamos el módulo de categorías

Esto nos llevara al módulo administrativo para las categorías de productos

91. Click en el botón "Nuevo"

Al hacer click nos desplegara una modal con el formulario para registrar una categoría

92. guardar con el formulario vacío

si se guarda con el formulario vacío se le pedirá que llene los campos que son obligatorios

93. Valores incorrectos

si en los campos se ingresa información que no esta permitida se mostrara el error y se le informara que información es valida

94. Llenar el campo Nombre de categoría

Tener en cuenta que el campo no permite caracteres especiales y tampoco permite números y tiene un limite de 50 caracteres

⛔️ El campo Nombre Categoría es obligatorio llenarlo

95. Llenar el campo Descripción

este campo tampoco permite caracteres especiales ni números y tiene un limite de 50 caracteres

Este campo no es obligatorio llenarlo, lo puedes dejar vacio si deseas :)

¿Cómo realizar el registro de una pérdida?

249. Click en nuevo

250. Ingresa el código de barras del producto

251. Este campo es obligatorio y no debe estar vacío

252. Este campo no admite carácteres especiales, ni espacios

253. Este campo admite valores alfanuméricos según corresponda a su código de barras

254. Ingresa la cantidad de productos que deseas registrar como una pérdida

255. Este campo es obligatorío y no puede estra vacío

256. Este campo no debe contener caráteres especiales, ni espacios

257. Ingresa el motivo por el cual deseas realizar el registro de la pérdida

258. Este campo es obligatorio y no debe estar vacío

259. Este campo solo admite valores alfabéticos, no admite carateres especiales ni tampoco números

260. Click en guardar

261. Ingresa la palabra clave que deseas buscar
